Class DataFrame

Description

The "DataFrame" class provides a class based Data Structure. Key Highlights:

  1. Store data in the form of Records with string keys and any values.
  2. Perform Statistical Operation like mean, mode, median, standard deviation , etc on numberic data
  3. Perform Data Manipulation Operations like filter, groupBy, aggregate, etc.
  4. Perform Advanced Data Manipulation Operations like merge,

Remarks

This class uses StatisticOperations (see StatisticOperations) for statistical operations , FileHandler (see FileHandler) for file handling operations and AdvancedOperations (see AdvancedOperations) for advanced operations.
